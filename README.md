# JD Automation

Herramientas internas para optimizar procesos operativos en SILA.

## Scripts

### Intake
Importación de Truck Log Excel a Pre Entrada SILA.

### Dispatch
Envío de órdenes a JD Flow Dispatch Board.

### Tracking + Docs
Tracking, hover de Orden de Carga y descarga rápida de documentos.

## Instalación

1. Instalar Tampermonkey.
2. Instalar cada script desde su enlace RAW.
3. No editar código directamente en Tampermonkey.

## Versionado

Cada cambio debe incrementar `@version`.

Ejemplo:
- 1.0 versión inicial
- 1.1 corrección menor
- 2.0 cambio grande
