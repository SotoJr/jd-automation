# Instalación JD Automation

## Requisitos

- Google Chrome
- Extensión Tampermonkey
- Acceso a SILA

---

## 1. Instalar Tampermonkey

Chrome Web Store:

https://www.tampermonkey.net/

---

## 2. Instalar JD Intake Lite

Abrir:

https://raw.githubusercontent.com/SotoJr/jd-automation/main/intake/jd-intake-lite.user.js

Seleccionar:

Install

---

## 3. Instalar Tracking + Docs

Abrir:

https://raw.githubusercontent.com/SotoJr/jd-automation/main/tracking/sila-tracking-docs.user.js

Seleccionar:

Install

---

## 4. Instalar Dispatch

Abrir:

https://raw.githubusercontent.com/SotoJr/jd-automation/main/dispatch/sila-enviar-dispatch.user.js

Seleccionar:

Install

---

## 5. Verificar instalación

Abrir Tampermonkey Dashboard.

Deben aparecer:

- JD Intake Lite - Truck Log Importer
- SILA Tracking + Docs
- SILA - Enviar a Dispatch

Todos en estado Enabled.
